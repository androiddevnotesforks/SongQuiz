Can you name the title or artist? Challenge yourself with arbitrary playlists!

Song Quiz guides you into the world of music. Play with more than 4 billion Spotify playlists to never make this game boring. Save playlists to your library, set up the game based on your preferences. Would it be too easy? Shorten the length of a song or disable repetition to make it more exciting.
